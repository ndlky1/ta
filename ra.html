使用說明：

請選擇以下任一方式載入您的招募與銷售資料：

方式一：透過 Apps Script API 自動同步 (推薦)

將您部署好的 Apps Script Web App 網址貼在下方，即可一鍵拉取最新資料。

方式二：手動上傳 CSV 檔案

上傳 Sheet 1 (新人資料)

包含 Batch(B), ID(C), HR/Ops(L), RJ/NJ(Q), Join Date(T), Last Date(AP)

上傳 Sheet 2 (銷售表現)

包含 Month(B), Staff ID(I), Actual Sales(N), Realized Sales(R)

尚未載入資料，請前往「資料來源設定」。
